This plugin will:
	•	Display a category dropdown and a Dropzone uploader on the page.
	•	Allow users to upload multiple images.
	•	Create a new post after all images are uploaded.
	•	Use the alt text of the first image as the post title.
	•	Display all uploaded images in the post content.
	•	Assign the selected category to the new post.
	•	Show an alert with a link to the newly created post.
